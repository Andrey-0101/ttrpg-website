# World of Darkness asset provenance

## Approved source

- Official source page: [World of Darkness Logos, Symbols and Free Assets Library](https://www.paradoxinteractive.com/games/world-of-darkness/community/free-content)
- Official download: the `WoD Community Assets` Dropbox folder linked by that Paradox page
- Source archive: `World of Darkness Community Assets.zip`, contained in the official folder download
- Download date: 2026-07-17
- Usage terms: [Dark Pack Agreement](https://www.paradoxinteractive.com/games/world-of-darkness/community/dark-pack-agreement)

No fan-made, traced, photographed, screenshot, or search-engine asset is used.

## Selected official files

The archive supplies transparent PNG dice faces and EPS/PNG standalone symbols, but no SVG versions of the rendered dice faces. The implementation uses the official transparent PNG dice faces so normal and Hunger styling remains exactly represented by the supplied assets.

## Confirmed display mapping

The numeric evaluator result remains authoritative. The presentation layer maps each supplied d10 value to an official visual category:

| Die kind | Numeric result | Official asset category |
|---|---:|---|
| Normal | 1–5 | Regular Failure |
| Normal | 6–9 | Regular Success |
| Normal | 10 | Regular Critical |
| Hunger | 1 | Hunger Bestial Failure |
| Hunger | 2–5 | Hunger Failure |
| Hunger | 6–9 | Hunger Success |
| Hunger | 10 | Hunger Messy Critical |
